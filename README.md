## 結論
本實驗成功驗證了 vLLM 前綴快取的效能。開啟快取後，系統能直接複用重複 Prompt 的 KV Cache，使總耗時從 3.20s 縮短至 1.66s，吞吐量大幅提升 1.92 倍。

雜湊表在快取檢索上的核心應用：透過雜湊將前綴映射至記憶體，在 $O(1)$ 的常數時間內精準命中並讀取既有特徵，省去重複計算的開銷。觀測數據完全符合投影片宣稱之「吞吐 +2~4x、延遲 -30~50%」量級，順利達成實驗目標。
